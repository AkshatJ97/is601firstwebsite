Purpose of this homework

The purpose of this homework is to learn basic linux commands Vi Text Editor, and Git from the command line.

Linux Commands

cd=change the directory you are working on mkdir name= make directory with filename cd filename=change directory to the new directory

VI Commands

vi filename=It opens the file that needs to be edited I=insert mode Escape=get out of the insert mode W=command to write Q=command to quit WQ=write and quit

Git Commands

Git init = Create the repository Git Add filename = add the file(s) to git in preparation for a commit Git commit -m “message”= do the commit to save changes Git checkout -b branch name = make a new branch and checkout Git checkout branch name = checkout branch Git merge BRANCHNAME = merge the branch
